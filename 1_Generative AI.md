** 1. What is Traditional AI?**

Traditional AI (also called Classical AI / Predictive AI) focuses on:
- Analyzing data and making predictions or decisions

Key Idea:
It answers:
  “What is this?”
  “What will happen?”

Examples:
Spam email detection
Fraud detection in banking
Image classification (cat vs dog)

**How it works:**
Uses structured data (tables, labeled datasets)
Trains models like:
  - Linear Regression
  - Decision Trees
  - Random Forest
  - SVM

Output:
Classification (Yes/No)
Prediction (numbers, probabilities)

AWS Autoscaling is example of Traditional AI.

**2. What is Generative AI?
**
Generative AI focuses on:
Creating new content (text, images, code, audio, etc.)

Key Idea:
It answers:
  - “Create something new”
  - “Generate content like this”

Examples:
Chatbots like ChatGPT
Image generation (DALL·E, Midjourney)
Code generation (GitHub Copilot)
Text summarization & translation

How it works:
Uses large datasets (text, images, code)

Trained using:
  - Neural Networks
  - Transformers (like GPT models)

Output:
Human-like text
Images
Code
Audio

Core Differences (Important for Interviews)
| Feature     | Traditional AI               | Generative AI               |
| ----------- | ---------------------------- | --------------------------- |
| Goal        | Predict / Classify           | Create new content          |
| Output      | Labels, numbers              | Text, images, code          |
| Data Type   | Structured data              | Unstructured data           |
| Examples    | Fraud detection, spam filter | Chatbots, image generation  |
| Models      | Regression, Trees, SVM       | Deep Learning, Transformers |
| Interaction | Limited                      | Highly interactive          |

----------------------

Traditional AI DevOps (MLOps)

 - Model training pipelines
